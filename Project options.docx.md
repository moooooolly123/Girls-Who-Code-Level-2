**Project 1: Personal Expense Tracker**

This project is a Python application to track personal expenses. It will allow the user to add, view, and categorize expenses, and provide summaries of spending over time. 

Week 1: Project Setup and Basic Functionality  
Goal: Set up the project structure and implement basic expense tracking.

- Create functions to add expenses (amount, category, date, description).

Week 2: Data Retrieval and Display  
Goal: Implement functions to retrieve and display expenses.

- Write functions to read expenses from user input.  
- Write function to display expenses in a formatted manner.

Week 3: Data Filtering  
Goal: Add basic data analysis filtering features  
Tasks: 

- Write functions that allow filtering expenses by date, category, amount.

Week 4: Data Analysis  
Goal: Add basic data analysis features.  
Tasks: 

- Write functions to calculate total expenses, average spending, category with highest expense any data analysis that might be important.

  

Week 5: Managing Savings  
Goal: Develop a saving management feature  
Tasks:

- Write a function to set a saving goal   
- Write a function to divide earnings based on saving goals

   
Week 6: Final Touches and Documentation  
Goal: Finalize the project and create documentation.  
Tasks:

-  Test the application thoroughly and fix any bugs.  
- Write documentation explaining how to use the application and its features.

**Project 2: Quiz Game**

This project is a Python quiz game where users answer questions and get scored.

Week 1: Project Setup and Basic Functionality  
Goal: Write a set of questions with their answers and display the questions randomly.  
Tasks:

- Create a small set of sample questions in a simple format, like a list of dictionaries. Each dictionary can represent a question and its answer  
- Write a function to display the questions randomly 

Week 2: Data Retrieval and Display  
Goal: Ask user quiz questions and provide feedback (points)  
Tasks:

- Write a function to obtain user’s answers to a question  
- Write a function to validate the answer and provide points to the user 

Week 3: Implement the teams    
Goal: Allow the game to  be played in teams   
Tasks:

- Write a function to let user’s join a team  
- Write a function to indicate which team’s turn is it   
- Modify point system to account for teams

Week 4: Add multiple choice answers  
Goal: Expand the game to ask multiple choice answers  
Tasks:

- Change your questions to a multiple-choice format. Include a list of options for each question.  
- Update your question display function to show the multiple-choice options.  
- Modify the input capture function  to handle the user selecting an option and validate the answer (give user points)

Week 5: Introduce different difficulty levels and question categories.  
Goal: Expand the game to ask questions from a specific category or difficulty  
Tasks:

- Organize your questions into categories (ex:  'Science', 'History').  
- Write a function for the user to choose a category before starting the quiz.  
- Modify your question display function to only ask questions from the chosen category.

Week 6: Final Touches and Documentation  
Goal: Finalize the project and create documentation.  
Tasks:

-  Test the application thoroughly and fix any bugs.  
- Write documentation explaining how to use the application and its features.
